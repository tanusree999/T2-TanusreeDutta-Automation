Step 1: Install Python in your system
Step 2: Install Google Chrome browser
Step 3: Download and set up ChromeDriver
Step 4: Clone this repository to your system
Step 5: Open the project in PyCharm
Step 6: Install Selenium using command pip install selenium
Step 7: Update login credentials inside the script Step
8: Run the Python script

This will execute the automation for login, generator creation, route assignment, service selection, price update and logout.
